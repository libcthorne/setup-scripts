Ubuntu setup script
=============
This script takes a fresh Ubuntu installation and installs applications, tools, and languages that I use.

Applications
* Chrome (stable and beta)
* VirtualBox
* Curl

Tools
* Git (with git config setup)

Languages and frameworks
* Ruby via rvm
* Rails
* Node.js
